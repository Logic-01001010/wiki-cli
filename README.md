# wiki-cli

## 권장
- 윈도우 환경

## 모듈설치
```
pip install -r requirements.txt
```

## 사용방법
```
python wiki.py "검색어" // 일반
python wiki.py -i "검색어" // 대화형
python wiki.py -h // 도움말
```
